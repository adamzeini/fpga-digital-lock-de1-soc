# Programmable Digital Lock System on Altera DE1-SoC FPGA

## Overview

This project is a programmable digital lock system implemented on the **Altera DE1-SoC FPGA** using **SystemVerilog**.

The system allows a user to enter a password using a keypad. The entered password is checked using hardware logic, and the system provides feedback through display output, buzzer indication, and servo motor control to simulate the lock/unlock mechanism.

This project was developed fully in hardware using SystemVerilog modules. No embedded C, NIOS processor, or JTAG-UART interface was used.

## Features

* Keypad-based password entry
* Password checking in hardware
* Failed-attempt lockout protection
* Password storage logic
* Password change control
* LCD/display feedback
* Buzzer alert indication
* Servo motor lock/unlock control
* Debounced button input
* Clock divider and timer modules
* Modular SystemVerilog design

## How It Works

1. The user enters a password using the keypad.
2. The keypad interface reads and processes each key press.
3. The password is stored and checked using SystemVerilog hardware logic.
4. If the password is correct, the system unlocks and activates the servo motor.
5. If the password is incorrect, the system gives an error indication.
6. After multiple failed attempts, the system enters a protected lockout state.
7. Password change logic allows updating the stored password through hardware control.

## Project Goal

The goal of this project is to design and implement a complete FPGA-based digital lock system using hardware logic. The project focuses on practical digital design concepts such as modular SystemVerilog development, finite state machines, keypad interfacing, debouncing, timing control, password verification, display control, and hardware-based output control.
